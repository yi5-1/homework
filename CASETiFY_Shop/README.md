# 脫殼 手機殼電商 - 第一階段

> 使用者可以瀏覽商品、加入購物車，無需登入即可暫存購物車

## 🎯 功能特性

### 前端功能
- ✅ **導覽列**：品牌 Logo、購物車圖標
- ✅ **分類橫條**：花卉、動物、食物、芭蕾風、抽象藝術
- ✅ **商品列表**：響應式網格佈局
- ✅ **訪客購物車**：使用 localStorage 本地存儲（無需登入）
- ✅ **購物車側邊欄**：查看、修改、刪除商品

### 後端 API
- `GET /` - 首頁
- `GET /api/products` - 獲取所有商品（按分類分組）

### 資料庫
- PostgreSQL 資料表：分類、商品

## 📦 安裝依賴

```bash
cd CASETiFY_Shop
pip install -r requirements.txt
```

## 🚀 啟動應用

### 1. 準備 PostgreSQL 資料庫

確保 PostgreSQL 服務已啟動，並記住連線資訊：
- Host (通常：localhost)
- Port (通常：5432)  
- Database (建議：casetify_db)
- User (通常：postgres)
- Password (你的 PostgreSQL 密碼)

### 2. 執行應用

```bash
python app.py
```

應用會提示你輸入資料庫連線資訊：
```
============================================================
 🔐 PostgreSQL 資料庫連線設定
============================================================
📍 Host (預設: localhost): [輸入 localhost 或 IP]
🔌 Port (預設: 5432): [輸入埠號]
💾 Database 名稱: [輸入資料庫名稱]
👤 使用者名稱 (預設: postgres): [輸入使用者名稱]
🔑 密碼: [輸入密碼]
```

### 3. 打開網站

在瀏覽器中開啟：`http://localhost:8000/`

## 🎨 使用說明

### 瀏覽商品
1. 點擊分類橫條的分類按鈕（花卉、動物等）
2. 查看該分類下的所有商品
3. 每個商品卡片顯示：名稱、顏色、價格

### 加入購物車
1. 點擊商品卡片上的「加入購物車」按鈕
2. 購物車徽章會顯示商品數量
3. 商品自動存儲到瀏覽器的 localStorage

### 查看購物車
1. 點擊頂部右側的購物車圖標 🛒
2. 側邊欄會滑出顯示購物車內容
3. 可以修改數量或刪除商品
4. 會自動計算總金額

## 💾 資料庫初始化

應用首次啟動時會自動執行 `schema.sql` 初始化資料庫，包括：
- 建立 `categories` 表（分類）
- 建立 `products` 表（商品）
- 插入 5 個默認分類
- 插入 13 個示例手機殼商品

## 🔒 安全性

- ✅ **資料庫連線資訊**不寫死在程式中
- ✅ 啟動時使用 `input()` 讓使用者手動輸入連線資訊
- ✅ 支援環境變數覆蓋（可選）
