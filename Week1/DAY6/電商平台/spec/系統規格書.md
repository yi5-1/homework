# 電商平台 — 系統規格書

## 1. 專案概述

輕量級電商平台，提供商品瀏覽、購物車、結帳與後台管理報表功能。

## 2. 技術架構

| 層級 | 技術 | 說明 |
|------|------|------|
| 前端 | HTML + CSS + JavaScript | 純靜態模板渲染 |
| 後端 | Python Flask (3.x) | RESTful API + 模板渲染 |
| 資料庫 | PostgreSQL | 關聯式資料庫 |
| 部署位址 | `http://localhost:8097` | 區域網路暴露 |

## 3. 功能頁面

| 頁面 | 路由 | 說明 |
|------|------|------|
| 首頁 | `/` | 平台形象展示、熱門商品、登入/註冊入口 |
| 商品頁 | `/products` | 列出平台上所有商品 |
| 購物車頁 | `/cart` | 顯示已加入的商品，可調整數量或移除 |
| 結帳頁 | `/checkout` | 從購物車送出訂單 |
| 後台登入 | `/admin/login` | 管理員專屬登入入口 |
| 後台儀表板 | `/admin/dashboard` | 會員統計、消費金額報表 |

## 4. 資料庫綱要

### users（使用者）
| 欄位 | 型態 | 備註 |
|------|------|------|
| id | SERIAL PRIMARY KEY | 自動流水號 |
| username | VARCHAR(50) UNIQUE NOT NULL | 使用者名稱 |
| email | VARCHAR(100) UNIQUE NOT NULL | 電子信箱 |
| password_hash | VARCHAR(255) NOT NULL | bcrypt 雜湊 |
| is_admin | BOOLEAN DEFAULT FALSE | 是否為管理員 |
| created_at | TIMESTAMP DEFAULT NOW() | 建立時間 |

### products（商品）
| 欄位 | 型態 | 備註 |
|------|------|------|
| id | SERIAL PRIMARY KEY | 自動流水號 |
| name | VARCHAR(200) NOT NULL | 商品名稱 |
| description | TEXT | 商品描述 |
| price | NUMERIC(10,2) NOT NULL | 單價 |
| stock | INTEGER DEFAULT 0 | 庫存量 |
| image_url | VARCHAR(500) | 商品圖片網址 |
| is_hot | BOOLEAN DEFAULT FALSE | 是否標記為熱門 |
| created_at | TIMESTAMP DEFAULT NOW() | 建立時間 |

### cart_items（購物車明細）
| 欄位 | 型態 | 備註 |
|------|------|------|
| id | SERIAL PRIMARY KEY | 自動流水號 |
| user_id | INTEGER REFERENCES users(id) | 使用者 FK |
| product_id | INTEGER REFERENCES products(id) | 商品 FK |
| quantity | INTEGER DEFAULT 1 | 數量 |
| created_at | TIMESTAMP DEFAULT NOW() | 加入時間 |

### orders（訂單）
| 欄位 | 型態 | 備註 |
|------|------|------|
| id | SERIAL PRIMARY KEY | 自動流水號 |
| user_id | INTEGER REFERENCES users(id) | 使用者 FK |
| total_amount | NUMERIC(12,2) NOT NULL | 訂單總金額 |
| status | VARCHAR(20) DEFAULT 'pending' | pending / paid / shipped / cancelled |
| created_at | TIMESTAMP DEFAULT NOW() | 建立時間 |

### order_items（訂單明細）
| 欄位 | 型態 | 備註 |
|------|------|------|
| id | SERIAL PRIMARY KEY | 自動流水號 |
| order_id | INTEGER REFERENCES orders(id) | 訂單 FK |
| product_id | INTEGER REFERENCES products(id) | 商品 FK |
| quantity | INTEGER NOT NULL | 數量 |
| price | NUMERIC(10,2) NOT NULL | 當下單價（快照） |

## 5. 資料流

```
瀏覽器 (HTML/CSS/JS)
    │
    ▼
Flask App (app.py)
    │
    ├── 模板渲染 → 回傳 HTML
    │
    └── API 路由 → JSON 回傳
            │
            ▼
        PostgreSQL
```

## 6. 安全注意事項

- 密碼採用 bcrypt 雜湊儲存，不存明碼
- session 使用 Flask 簽名 cookie
- .env 包含資料庫帳密與 SECRET_KEY，已加入 .gitignore
- 後台路由需要 session 驗證管理員身份
